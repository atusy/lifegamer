# lifegamer


Lifegame implementation in R without `if`-expressions and `for`-loops.\`

``` r
animate(lifegame(first_generation("segment10", 20, 30), 25), ncol = 5)
```

``` r
animate(lifegame(first_generation("octagon", 20, 30), 10), ncol = 5)
```
